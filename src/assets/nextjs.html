This open source icon is named "nextjs" and is licensed under the open source MIT license. It's available to be downloaded in SVG and PNG formats (available in 256, 512, 1024 and 2048 PNG sizes).

It's part of the icon set "Teenyicons", which has 1,200 icons in it.

If you need this icon available in another format, it should be pretty straight forward to download it as an SVG image file, and then import it into apps like Canva, Fotor, Illustrator or Pixlr.
